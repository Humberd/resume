> The project is my Resume written in React.
It is print-optimized, so you can print it either physically or to pdf
without any layout bugs.

## Install

```sh
npm ci
```

## Usage

```sh
npm start
```

## Generate PDF

Before generating PDF make sure the server is running on port 8000.

```sh
npm run pdf
```
